# Kolibrimini
Kolibrimini is 40% Alice-style keyboard with arrow cluster
- Uses Pro Micro 
- Plateless
- Pcb isolated from case using rubber plugs and o-rings
- MJF printed case (JLCPCB)
- Rgb underglow
